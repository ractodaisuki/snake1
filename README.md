# snake1

Pyxel で作ったシンプルな Snake ゲームです。

## GitHub Pages

GitHub Pages を有効化すると、`main` への push ごとに自動で公開されます。

公開 URL:

`https://ractodaisuki.github.io/snake1/`

最初の 1 回だけ GitHub のリポジトリ設定で `Settings > Pages > Build and deployment > Source` を `GitHub Actions` にしてください。

## 起動

`index.html` をブラウザで開くと、Pyxel Web 版で `snake1.py` が起動します。

## 操作

- 矢印キー / `WASD`: 移動
- `Space` / `Enter`: スタート、リトライ
- `Esc`: 終了
